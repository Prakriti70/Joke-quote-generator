# Joke & Quote Generator

This is a small Python practice project for generating random jokes and quotes, created to practice basic Python programming skills.

## Features

- Shows a random joke or quote each time it runs.
- Stores jokes and quotes in an external text file for easy editing and expansion.

---

## Requirements

- Python 3.x


## How to Run

1. Clone or download the repository.
2. Make sure `jokes_quotes.txt` is in the same folder as the Python script.
3. Run the program:

```bash
python joke_quote_generator.py
